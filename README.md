# picgo-plugin-confluence
支持上传到Confluence图库

PicGo Uploader For confluence

### Install

```bash
npm i picgo-plugin-confluence
```

### Usage
- CONFLUENCE_BASE_URL:
- username: confluence's username
- USER_PASSWORD: confluence's password
- ATTACHMENT_TYPE: 
- ATTACHMENT_FILE_NAME: 
- PAGE_ID
